# Backend de Agentemotor

Directorio `backend/` contiene la implementación del servidor Express con SQLite.

## Comandos

- `npm install`
- `npm run dev`
- `npm test`

## Notas

- `src/` contiene la aplicación y las rutas.
- `tests/` contiene pruebas de unidad.
- Use `.env.example` para crear variables de entorno.
